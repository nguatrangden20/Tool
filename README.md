# Tool
ToolEncrypt
Đây là Tool dùng để mã hóa file bằng thuật toán đảo ngược bit,

Có các chức năng như:
- Chọn file từ thư mục 
- Hiển thị danh sách file
- Hiển thị thông tin file [hiện tại chỉ hỗ trợ 2 đuôi txt và png]
- Sử dụng Thread để Encrypt.
- Nén lại file sau khi đã mã hóa xong
- Tạo file Meta
- Một số tùy chỉnh

Tương lai:
- Optimize và Clean lại Code
- Hỗ trợ đa nền tảng [Hiện tại chỉ chạy trên Windows]
